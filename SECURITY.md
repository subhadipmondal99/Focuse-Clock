# 🔐 Security Policy

## 📌 Supported Versions

This project is actively maintained. Security updates are provided for the latest version only.

| Version        | Supported |
| -------------- | --------- |
| Latest         | ✅ Yes     |
| Older Versions | ❌ No      |

---

## 🚨 Reporting a Vulnerability

If you discover any security vulnerability in this project, please report it responsibly.

### 📩 How to Report

* Open a **private issue** on GitHub (if possible), or
* Contact directly via LinkedIn:
  👉 https://www.linkedin.com/in/subhadip--mondal

---

## ⏱️ Response Time

* Initial response: within **48–72 hours**
* Fix and update: depends on severity, usually within **a few days**

---

## 🔍 What to Expect

After reporting:

* Your report will be reviewed carefully
* You may be contacted for more details
* If valid, the issue will be fixed and updated
* You will be acknowledged for your contribution

---

## ❗ Scope

This is a **frontend-only project**, so major security risks are minimal. However, the following are still considered:

* Cross-site scripting (XSS)
* Dependency vulnerabilities (if added in future)
* Misuse of browser APIs (like Web Audio API)

---

## ⚠️ Disclaimer

This project does not handle:

* User authentication
* Sensitive personal data
* Payment or financial information

---

## 🤝 Responsible Disclosure

Please do not publicly disclose vulnerabilities until they are resolved. Responsible reporting helps keep the project safe for everyone.

---

Thank you for helping improve the security of this project 🙏
